  • [issue 24] Select a different html template file while composer is open. Note that any manually entered text will be discarded when you change the template, so do this before you start typing. In order to do this the new Change Template button can be added via right-click and Customize on the composer toolbar. After adding the button, reopen the Composer window to update it with the appropriate template list (write / reply or forward).
  • [issue 60] Fixed some barriers with screenreader navigation in Settings Dialog. All buttons on the settings screen can now be executed without mouse clicks. Also the licensing message will be displayed as a OS notification which will be read out by the widely accepted JAWS reader. It appears that NVDA still cannot read these.
